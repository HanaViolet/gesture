# NASA-TLX 认知负荷评估量表

## 使用说明

本量表用于评估您在使用小程序时的认知负荷水平。请根据您的实际体验，在0-100之间为每个维度评分。

**评分标准**：
- 0 = 非常低/非常好
- 25 = 低/好
- 50 = 中等
- 75 = 高/差
- 100 = 非常高/非常差

**说明**：
- 低 = 任务很简单，不需要太多思考
- 高 = 任务很复杂，需要高度集中注意力、大量思考

**说明**：
- 低 = 几乎不需要体力活动（如点击、滑动很轻松）
