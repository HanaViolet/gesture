## 一、应用简介

**语见**是一款专注于无障碍沟通的小程序，帮助听障人士和普通用户更好地进行交流。

### 主要功能

| 功能 | 说明 |
|------|------|
| 🎯 双模式切换 | 普通版 / 听障版，满足不同用户需求 |
| 🤟 手语翻译 | 手语识别（视频→文字）/ 手语生成（文字→动画） |
| 🔊 语音合成 | 文字转语音，支持男女声选择 |
| 💚 心理咨询 | AI心理健康助手，支持手语对话 |
| 🏢 社区工作 | 分享经验、寻求帮助、交流互动 |

**【图片占位：普通版首页截图】**

#### 文字转手语
1. 在输入框中输入想要转换为手语的文字
2. 点击"生成手语"按钮

系统会将手语动作识别为文字显示。

**【图片占位：手语识别结果截图】**

## 四、手语翻译

### 4.1 手语识别（视频→文字）

**【图片占位：手语识别流程截图】**

1. 点击首页的"相机"按钮或"手语识别"卡片
2. 选择"拍摄手语"或"从相册上传"
3. 录制或选择手语视频（建议3-10秒）
4. 系统自动识别，显示文字结果
5. 可以播放男声/女声语音

**【图片占位：识别结果页面截图】**

### 4.2 手语生成（文字→动画）

**【图片占位：手语生成流程截图】**

1. 在输入框中输入文字
2. 点击"生成手语"按钮
3. 等待系统生成（约10-30秒）
4. 查看生成的手语动画视频

**【图片占位：手语动画播放截图】**

### 4.3 保存到常用语

识别或生成的结果可以保存到常用语：

1. 在结果页面点击"保存到常用语"
2. 内容将出现在首页快捷短语中

**【图片占位：保存常用语截图】**

3. 用手语表达您的感受
4. 系统识别后自动发送给AI助手
5. AI助手会理解并回应

每条AI回复下方都有"🤟 查看手语"按钮：
1. 点击按钮
2. 系统生成手语动画
3. 播放手语视频

## 八、常见问题

### Q1：手语识别不准确怎么办？

**【图片占位：FAQ截图】**

- 确保光线充足，背景简洁
- 手语动作清晰、速度适中
- 尽量正对相机，避免侧面角度
- 视频时长建议在3-10秒

### Q2：语音合成没有声音？

- 检查手机是否静音
- 检查小程序是否有音频播放权限
- 尝试调整语速设置

### Q3：如何切换语言？

进入设置 → 通用设置 → 语言，选择您需要的语言。

### Q4：心理咨询助手的对话会被保存吗？

对话历史会自动保存在本地，您可以在设置中清空历史记录。

### Q5：生成的手语动画可以保存吗？

目前手语动画暂不支持下载保存，但可以随时重新生成。
