## 简介

**语见**是一款专为无障碍沟通打造的AI手语双向翻译与生成微信小程序，致力于搭建听障人士与健听人群之间的沟通桥梁。

依托先进的人工智能技术，语见构建了从手语到语音、从语音到手语的双向无障碍沟通链路，为弥合不同语言群体之间的交流鸿沟提供一站式的技术解决方案。

## 功能特性

### 核心功能

| 功能 | 描述 | 适用场景 |
|:----:|:-----|:---------|
| 🤟 **手语识别** | 将手语视频实时转换为文字/语音 | 听障人士与不懂手语的人交流 |
| 🎬 **手语生成** | 将文字/语音转换为3D手语动画 | 健听人学习手语、与听障朋友沟通 |
| ⚡ **快捷短语** | 一键发送常用语句，支持自定义 | 高频场景快速表达 |
| 💬 **AI对话助手** | 基于大模型的情感陪伴与对话 | 随时随地的智能陪伴 |

### 双模式设计

- **听障模式**：大按钮、大字体、高对比度，操作简洁明了
- **普通模式**：信息密度高，支持语音输入和手语学习

### 国际化支持

支持简体中文、English、Deutsch三种语言界面。

## 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                      语见小程序                          │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────┐ │
│  │  听障模式    │  │  普通模式    │  │    AI对话助手    │ │
│  └─────────────┘  └─────────────┘  └─────────────────┘ │
└──────────────────────────┬──────────────────────────────┘
                           │
┌──────────────────────────▼──────────────────────────────┐
│                     主服务 API                           │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────┐ │
│  │  Uni-Sign   │  │   ChatTTS   │  │  Qwen (微调)    │ │
│  │  手语识别   │  │  语音合成    │  │  AI对话引擎      │ │

                           │
```

### 核心技术栈

- **前端**：微信小程序原生开发
- **3D渲染**：Three.js
- **手语识别**：Uni-Sign 统一生成式框架
- **手语生成**：SOKE 技术框架
- **语音合成**：ChatTTS
- **AI对话**：Qwen 模型微调

## 快速开始

### 环境要求

- 微信开发者工具
- Node.js >= 16.x

### 安装运行

1. 克隆仓库
```bash
git clone https://github.com/HanaViolet/gesture.git
cd gesture
```

2. 使用微信开发者工具打开项目

3. 配置后端API（`utils/api.js`）
```javascript
const CURRENT_ENV = ENV.PRODUCTION; // 或 ENV.DEVELOPMENT
```

4. 编译运行
