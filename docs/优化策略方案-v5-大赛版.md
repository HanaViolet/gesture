## V1.0 核心功能稳定版

### 目标
双模式翻译功能完全可用，SMPL手语渲染与TTS语音调用稳定，无阻塞性BUG。

### 功能清单
- [ ] 正常模式：输入文字/语音 → 调用SMPL → 播放手语动画
- [ ] 聋人模式：输入文字 → 调用TTS API → 播放语音
- [ ] 双模式各保留3条最近记录（本地存储）
- [ ] 语言切换：zh/en/de 全局生效
- [ ] 输入为空时的友好提示

### 技术要点
- SMPL加载状态管理（loading → 渲染 → 播放控制）
- TTS错误处理（网络失败、服务未启动的降级提示）
- 历史记录LRU淘汰机制

### 验收标准
- 正常模式端到端流程 < 3秒
- 聋人模式语音合成 < 2秒
- 连续使用10次无崩溃

## V2.0 视觉高级感重塑

### 目标
页面呈现克制高级感，Morandi色系贯穿，微交互动画细腻。

### 设计规范

#### 色彩系统
```
主色: #7A8B99 (灰蓝，沉稳可信)
辅助: #D4A373 (琥珀，温暖点缀)
背景: #F5F3F0 (米灰，纸质感)
卡片: #FFFFFF (纯白，悬浮感)
文字主: #2C3E50 (深灰蓝)
文字次: #7A8B99 (中灰)
分割线: rgba(122,139,153,0.1)
```

#### 字体规范
- 主标题: 20px, weight 600
- 副标题: 16px, weight 500
- 正文: 14px, weight 400
- 辅助: 12px, weight 400

#### 圆角体系
- 大卡片: 24rpx
- 按钮: 16rpx
- 输入框: 12rpx
- 小标签: 8rpx

#### 阴影层级
```css
/* 悬浮卡片 */
box-shadow: 0 4rpx 20rpx rgba(122,139,153,0.08);

/* 按钮按下 */
box-shadow: 0 2rpx 8rpx rgba(122,139,153,0.12);

/* 强调元素 */
box-shadow: 0 8rpx 32rpx rgba(212,163,115,0.15);
```

### 动效设计
- 页面切换: 淡入 200ms, ease-out
- 按钮反馈: scale(0.98) + 背景变深, 100ms
- 卡片悬停: 轻微上浮 translateY(-2rpx)
- 历史记录滑入: stagger 50ms

### 页面重构重点
1. **主页**: 双模式切换动效、输入区呼吸感、历史记录卡片化
2. **社区**: 瀑布流简化、点赞动效、发布按钮悬浮
3. **设置**: 分组清晰、开关细腻、指引可折叠

## V4.0 无障碍深度适配

### 目标
符合WCAG 2.1 AA标准，让任何用户都能顺畅使用。

### 适配清单

#### 视觉无障碍
- [ ] 颜色对比度 ≥ 4.5:1（所有文字）
- [ ] 支持系统字体大小调整（110%-150%适配）
- [ ] 焦点状态清晰可见（高亮边框）

- [ ] 手势操作有替代方案（按钮）

- [ ] 图片有意义的alt文本
- [ ] 表单标签与输入框关联

### 测试检查项
- [ ] 色盲模拟器测试通过
- [ ] 最大字体不重叠截断
- [ ]  VoiceOver/TalkBack可读

## V5.0 演示体验优化版

### 目标
大赛演示场景优化，首屏惊艳，流程顺畅，容错友好。

### 优化点

#### 1. 启动体验
- 启动页：品牌Logo淡入 + Slogan"连接无声与有声"
- 首屏预加载：SMPL模型静默加载
- 新手引导：首次使用3步轻引导（可跳过）

#### 2. 演示模式
- **演示开关**：设置中隐藏，开启后：
  - SMPL使用预置动画（不依赖后端）
  - TTS使用模拟播放（显示波形动画）
  - 历史记录预置3条示例数据

#### 3. 容错设计
- 网络断开：本地缓存上次输入，恢复后自动重试
- TTS服务未启动：友好提示"请启动语音服务"
- SMPL加载失败：显示静态示意图+重试按钮

#### 4. 性能兜底
- 首屏渲染 < 1.5秒
- 内存占用监控，超出提示重启
- 页面切换无白屏

#### 5. 演示数据
- 预设3条高质量社区帖子
- 预设3条翻译历史记录
- 设置页展开所有功能展示

## 技术实现要点

### 目录结构
```
components/
  ├── smpl-viewer/      # SMPL渲染组件
  ├── tts-player/       # TTS控制组件
  ├── mode-switch/      # 模式切换组件
  ├── history-card/     # 历史记录卡片
  └── community-card/   # 社区帖子卡片

utils/
  ├── smpl-loader.js    # SMPL模型管理
  ├── tts-client.js     # TTS API封装
  ├── storage.js        # 本地存储工具
  └── accessibility.js  # 无障碍辅助

pages/
  ├── home/             # 主页（双模式）
  ├── community/        # 社区
  ├── post-detail/      # 帖子详情
  └── settings/         # 设置
```

### 关键依赖
- three.js: SMPL手语渲染
- 微信原生API: TTS调用、文件上传

### 性能优化
- SMPL模型懒加载+缓存
- 图片懒加载+预加载策略
- 列表虚拟滚动（社区帖子>20条）
- 节流防抖处理（输入、滚动）
