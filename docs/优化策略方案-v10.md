## 一、项目现状与需求回顾

### 1.1 产品定位
**语见**是一款专注于无障碍沟通的小程序，核心定位是：
- **普通用户** → 输入文字 → 生成SMPL手语动画（给听障人看）
- **听障用户** → 输入文字/手语 → 生成语音/文字（给普通人听/看）

### 1.2 当前已实现功能
| 模块 | 功能状态 |
|------|---------|
| 双模式首页 | ✅ 普通版/听障版切换 |
| SMPL手语动画 | ✅ Three.js本地渲染 |
| TTS语音合成 | ✅ 本地API调用 |
| 手语识别 | ✅ 视频上传识别（预留AI接口） |
| 社区论坛 | ✅ 发帖/评论/点赞（预留后端API） |
| 国际化 | ✅ 中/英/德三语言 |
| 无障碍设置 | ✅ 5档字体/高对比度/减少动画 |
| 使用记录 | ✅ 双模式各3条历史记录 |

### 1.3 核心需求总结
1. **普通版**：SMPL使用记录（最近3次），非对话记录
2. **听障版**：输出显示框+历史记录，支持手语识别
3. **社区**：真实发帖功能，预留后端接口
4. **品牌**：已更名"语见"，需持续强化
5. **设计**：莫兰迪色系，高级感+无障碍
