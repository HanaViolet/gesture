# 语见App用户手册

## 欢迎使用语见

语见是一款专为无障碍沟通打造的智能应用，致力于搭建听障人士与健听人群之间的沟通桥梁。在这个充满温度的数字空间里，每一次交流都应该被温柔以待。无论您是需要手语翻译的听障朋友，还是希望学习手语的健听人士，语见都将陪伴您走过每一步沟通旅程。

我们的核心理念是"打破沉默，连接心灵"。通过先进的AI技术和人性化的交互设计，语见让沟通不再有障碍，让每一个声音都能被听见，让每一个手势都能被理解。

### 2.1 顶部导航区域

进入语见的主界面，您首先看到的是精心设计的顶部导航栏。左侧是语见的品牌标识，那个温暖的图标象征着沟通的桥梁。中间是模式切换按钮，采用胶囊式的设计，您可以在这里随时在普通模式和听障模式之间切换。右侧则是设置入口，点击齿轮图标即可进入设置页面。

## 第三章 手语翻译功能

### 3.1 手语识别流程

手语识别是语见的核心功能之一，它能够将您的手语动作准确转换为文字。整个识别流程设计得非常简单直观。

当您点击首页的相机按钮后，系统会弹出选项菜单，询问您是从相册选择视频还是直接拍摄。如果您选择拍摄，会进入专门的相机页面。这个页面的设计充分考虑了听障用户的使用习惯，界面简洁，操作按钮都非常醒目。

【📷 截图占位：相机拍摄页面 - 请在此处放置相机拍摄界面的截图，展示录制按钮和倒计时】

在相机页面，大圆形的录制按钮占据了视觉中心。点击开始后，系统会开始录制您的手语动作，同时显示30秒的倒计时。录制过程中，按钮会变成红色并显示波纹效果，让您清楚地知道正在录制中。录制完成后，再次点击按钮即可停止录制。

【📷 截图占位：录制中状态 - 请在此处放置正在录制状态下的界面截图】

录制完成的视频会自动上传到后端服务器进行识别处理。这个过程通常需要几秒钟的时间，期间会显示加载动画。识别完成后，结果会以文字形式展示在结果页面，同时系统还会生成对应的语音，您可以选择男声或女声进行播放。

【📷 截图占位：识别结果页面 - 请在此处放置分析结果页面的截图，展示识别出的文字和语音播放按钮】

### 3.2 手语动画生成

与手语识别相反，手语动画生成是将文字转换为可视化的手语动作。这项功能采用了先进的SOKE技术，能够生成流畅自然的3D人体动画。

在普通版首页输入文字并点击生成后，系统会创建一个生成任务。由于生成过程需要一定的计算时间，我们会展示一个精美的圆形进度环，实时显示当前的生成进度。进度环采用渐变色彩设计，配合百分比数字显示，让您清楚地了解生成状态。您可以在这个过程中选择等待，或者点击取消按钮终止生成。

【📷 截图占位：手语动画生成进度 - 请在此处放置生成进度页面的截图，展示圆形进度环和百分比】

当生成完成后，3D手语动画会在全屏播放器中展示。播放器采用居中对齐的布局，视频区域位于屏幕中央，确保最佳观看体验。播放器提供了多种控制选项：您可以调整播放速度，从0.5倍速到1.5倍速共有五档可选；可以暂停和重新播放；还可以将生成的视频分享给您的朋友。如果您觉得动画速度太快跟不上，建议使用0.75倍速慢慢学习。

## 第六章 常见问题解答

在使用语见的过程中，您可能会遇到一些疑问。以下是用户最常咨询的几个问题。

**关于手语识别的准确度**，这取决于多种因素，包括拍摄时的光线条件、背景是否杂乱、手势是否标准等。建议在光线充足、背景简洁的环境下进行拍摄，同时尽量使用标准的中国手语动作，这样可以获得最佳的识别效果。

**如果您发现语音播放没有声音**，请首先检查手机的媒体音量是否开启，然后确认是否授予了应用音频播放权限。在iOS系统中，还需要检查是否开启了静音模式。

**手语动画生成需要依赖网络连接**，因为它需要将您的请求发送到云端服务器进行处理。建议在WiFi环境下使用此功能，以获得更快的生成速度。如果长时间显示生成中，可能是网络连接不稳定，您可以尝试取消后重新生成。

**快捷短语的自定义数量上限是15条**，这是为了保证界面的整洁和操作的便捷。如果您已经达到了上限，需要先删除一些不常用的短语，才能添加新的内容。

**心理咨询功能中的AI助手**是基于大规模语言模型训练的，它能够提供情感支持和心理疏导，但并不能替代专业的心理医生。如果您有严重的心理健康问题，建议及时寻求专业的医疗帮助。
