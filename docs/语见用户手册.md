# 语见App用户手册

## 欢迎使用语见

**"无声世界，语见其声"**

语见是一款专为无障碍沟通打造的AI手语双向翻译与生成系统，致力于搭建听障人士与健听人群之间的沟通桥梁。在这个充满温度的数字空间里，每一次交流都应该被温柔以待。

依托先进的人工智能技术，语见构建了从手语到语音、从语音到手语的**双向无障碍沟通链路**，为弥合不同语言群体之间的交流鸿沟提供一站式的技术解决方案。无论您是需要手语翻译的听障朋友，还是希望学习手语的健听人士，语见都将陪伴您走过每一步沟通旅程。

## 第一章 初识语见

### 1.1 产品简介

语见的核心理念是**"打破沉默，连接心灵"**。通过以下四大核心功能，我们让沟通不再有障碍：

- **手语识别**：听障人士通过摄像头拍摄手语，系统实时转换为文字并语音播报
- **手语生成**：健听人士输入文字或语音，系统翻译为三维虚拟人手语动画
- **快捷短语**：内置常用语快捷库，支持高频场景快速调用
- **心理助手**：面向聋哑群体的情感支持智能体，提供情感陪伴与心理疏导

## 第三章 手语翻译功能

### 3.1 手语识别流程

手语识别是语见的核心功能之一，依托**Uni-Sign统一生成式框架**，将多种手语识别任务统一建模为序列生成问题，并融合姿态与RGB多模态信息以增强细粒度感知能力。

#### 拍摄与上传
当您点击首页的相机按钮后，系统会弹出选项菜单，询问您是从相册选择视频还是直接拍摄。如果您选择拍摄，会进入专门的相机页面。这个页面的设计充分考虑了听障用户的使用习惯，界面简洁，操作按钮都非常醒目。

【📷 截图占位：相机拍摄页面 - 请在此处放置相机拍摄界面的截图，展示录制按钮和倒计时】

在相机页面，大圆形的录制按钮占据了视觉中心。点击开始后，系统会开始录制您的手语动作，同时显示30秒的倒计时。录制过程中，按钮会变成红色并显示波纹效果，让您清楚地知道正在录制中。录制完成后，再次点击按钮即可停止录制。

【📷 截图占位：录制中状态 - 请在此处放置正在录制状态下的界面截图】

#### 识别结果
录制完成的视频会自动上传到后端服务器进行识别处理。这个过程通常需要几秒钟的时间，期间会显示加载动画。识别完成后，结果会以文字形式展示在结果页面，同时系统还会生成对应的语音，您可以选择男声或女声进行播放。

【📷 截图占位：识别结果页面 - 请在此处放置分析结果页面的截图，展示识别出的文字和语音播放按钮】

### 3.2 手语动画生成

与手语识别相反，手语动画生成是将文字转换为可视化的手语动作。这项功能采用了先进的**SOKE技术框架**，通过将连续三维运动离散化为动作符号，使大语言模型能够自回归地预测手语动作序列，生成流畅自然的3D人体动画。

#### 生成进度
在普通版首页输入文字并点击生成后，系统会创建一个生成任务。由于生成过程需要一定的计算时间，我们会展示一个精美的**圆形进度环**，实时显示当前的生成进度。进度环采用渐变色彩设计，配合百分比数字显示，让您清楚地了解生成状态。

【📷 截图占位：手语动画生成进度 - 请在此处放置生成进度页面的截图，展示圆形进度环和百分比】

#### 动画播放
当生成完成后，3D手语动画会在全屏播放器中展示。播放器采用**居中对齐**的布局，视频区域位于屏幕中央，确保最佳观看体验。播放器提供了多种控制选项：

- **播放速度调节**：从0.5倍速到1.5倍速共有五档可选，0.75倍速特别适合初学者跟练
- **暂停与重播**：随时控制播放进度
- **分享功能**：将生成的视频分享给朋友

## 第六章 常见问题解答

**Q：手语识别的准确度如何？**

A：准确度取决于多种因素，包括拍摄时的光线条件、背景是否杂乱、手势是否标准等。建议在光线充足、背景简洁的环境下进行拍摄，同时尽量使用标准的中国手语动作。我们的Uni-Sign框架融合了姿态与RGB多模态信息，能够有效提升识别准确率。

**Q：语音播放没有声音怎么办？**

A：请首先检查手机的媒体音量是否开启，然后确认是否授予了应用音频播放权限。在iOS系统中，还需要检查是否开启了静音模式。

**Q：手语动画生成需要联网吗？**

A：是的，手语动画生成需要依赖网络连接，因为涉及到复杂的3D渲染计算，需要将您的请求发送到云端服务器进行处理。建议在WiFi环境下使用此功能，以获得更快的生成速度。

**Q：快捷短语的数量有限制吗？**

A：快捷短语的自定义数量上限是15条，这是为了保证界面的整洁和操作的便捷。如果您已经达到了上限，需要先删除一些不常用的短语，才能添加新的内容。

**Q：心理咨询功能中的AI助手可靠吗？**

A：AI助手是基于Qwen模型微调训练的情感支持智能体，在情感支持专用对话数据上进行了适配训练。它能够提供情感支持和心理疏导，但并不能替代专业的心理医生。如果您有严重的心理健康问题，建议及时寻求专业的医疗帮助。
